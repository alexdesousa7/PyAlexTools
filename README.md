📌 Año 2026 — Mi regreso al proyecto

Después de **cinco años** sin programar en Python, he decidido retomar este proyecto con una mentalidad nueva, más madura y con muchas ganas de aprender y crear herramientas útiles.

Este repositorio incluira desde scripts simples creados en 2021 hasta interfaces gráficas completas, proyectos retro y aplicaciones organizadas con Programación Orientada a Objetos.

Mi objetivo es seguir ampliando este repositorio con nuevas versiones, nuevas herramientas y proyectos más complejos.
